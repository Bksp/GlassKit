Guía de Uso: GlassKit

Aprende a integrar, estructurar y potenciar tus interfaces con la arquitectura Glassmorphism de GlassKit. Los diagramas interactivos ilustran el funcionamiento interno del framework.

Instalación Base

GlassKit funciona como una envoltura (wrapper) sobre Bootstrap 5. Inyecta dinámicamente las variables CSS necesarias para lograr el efecto cristalizado.

Importa main.js como módulo en tu HTML para activar el motor de temas y el enrutador SPA.

Arquitectura del Framework

graph TD
    A[index.html] --> B(GlassKit Core)
    B --> C{Motor CSS Variables}
    C -->|Inyecta Colores| D[Bootstrap 5]
    C -->|Actualiza SVGs| E[Mermaid.js]
    B --> F[Efectos Glassmorphism]

Utilizando Clases

Añade la clase glass-card a cualquier contenedor. Usa utilidades como btn-glass-primary para botones esmerilados y bg-opacity-10 para paneles translúcidos.

sequenceDiagram

    TM->>DOM: Sobrescribe --bs-primary, --bg-color
    TM-->>M: Evento: glasskit-theme-changed
    activate M

    deactivate M
